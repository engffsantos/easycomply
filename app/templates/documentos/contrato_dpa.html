1. DEFINIÇÕES

Para os fins deste DPA, os termos abaixo terão os seguintes significados:

1.1. "Dados Pessoais": informação relacionada a pessoa natural identificada ou identificável;

1.2. "Titular": pessoa natural a quem se referem os dados pessoais que são objeto de tratamento;

1.3. "Tratamento": toda operação realizada com dados pessoais, como coleta, produção, recepção, classificação, utilização, acesso, reprodução, transmissão, distribuição, processamento, arquivamento, armazenamento, eliminação, avaliação ou controle da informação, modificação, comunicação, transferência, difusão ou extração;

1.4. "LGPD": Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018);

1.5. "Incidente de Segurança": qualquer evento adverso confirmado, relacionado à segurança dos dados pessoais, que possa comprometer a confidencialidade, integridade ou disponibilidade dos dados pessoais.

2. OBJETO

2.1. O presente DPA tem por objeto estabelecer as condições, obrigações e responsabilidades das Partes quanto ao tratamento de dados pessoais realizado pelo OPERADOR em nome do CONTROLADOR, em conformidade com a LGPD.

2.2. O OPERADOR realizará o tratamento dos seguintes dados pessoais em nome do CONTROLADOR:

{{ tipo_dados }}

3. FINALIDADE DO TRATAMENTO

3.1. O OPERADOR tratará os dados pessoais exclusivamente para as seguintes finalidades:

{{ finalidade }}

3.2. O OPERADOR não poderá utilizar os dados pessoais para qualquer outra finalidade que não as expressamente autorizadas pelo CONTROLADOR neste DPA.

4. OBRIGAÇÕES DO OPERADOR

4.1. O OPERADOR compromete-se a:

a) Tratar os dados pessoais de acordo com as instruções documentadas do CONTROLADOR;

b) Garantir que as pessoas autorizadas a tratar os dados pessoais assumam um compromisso de confidencialidade;

c) Adotar medidas técnicas e organizativas de segurança apropriadas para proteger os dados pessoais;

d) Não contratar outro operador sem autorização prévia do CONTROLADOR;

e) Auxiliar o CONTROLADOR no atendimento às solicitações dos titulares de dados;

f) Auxiliar o CONTROLADOR no cumprimento das obrigações relativas à segurança dos dados, notificação de incidentes e avaliação de impacto;

g) Eliminar ou devolver todos os dados pessoais ao CONTROLADOR após o término da prestação de serviços;

h) Disponibilizar ao CONTROLADOR todas as informações necessárias para demonstrar o cumprimento das obrigações previstas neste DPA.

5. OBRIGAÇÕES DO CONTROLADOR

5.1. O CONTROLADOR compromete-se a:

a) Fornecer instruções documentadas ao OPERADOR sobre o tratamento de dados pessoais;

b) Garantir que possui base legal para o tratamento dos dados pessoais;

c) Informar o OPERADOR sobre quaisquer alterações nas instruções de tratamento;

d) Responder às solicitações dos titulares de dados.

6. SEGURANÇA DOS DADOS

6.1. O OPERADOR implementará medidas técnicas e organizativas adequadas para garantir um nível de segurança apropriado ao risco, incluindo, entre outras:

a) Pseudonimização e criptografia de dados pessoais;

b) Capacidade de assegurar a confidencialidade, integridade, disponibilidade e resiliência dos sistemas e serviços de tratamento;

c) Capacidade de restabelecer a disponibilidade e o acesso aos dados pessoais em tempo hábil no caso de um incidente físico ou técnico;

d) Processo para testar, avaliar e verificar regularmente a eficácia das medidas técnicas e organizativas para garantir a segurança do tratamento.

7. NOTIFICAÇÃO DE INCIDENTES

7.1. O OPERADOR notificará o CONTROLADOR, sem demora injustificada, após tomar conhecimento de um incidente de segurança que afete os dados pessoais tratados em nome do CONTROLADOR.

7.2. A notificação deverá conter, no mínimo:

a) Descrição da natureza do incidente;

b) Categorias e número aproximado de titulares afetados;

c) Categorias e número aproximado de registros de dados afetados;

d) Medidas adotadas ou propostas para mitigar os possíveis efeitos adversos.

8. SUBCONTRATAÇÃO

8.1. O OPERADOR não poderá subcontratar outro operador para realizar atividades de tratamento específicas em nome do CONTROLADOR sem autorização prévia e específica do CONTROLADOR.

8.2. Quando o OPERADOR recorrer a outro operador para realizar atividades de tratamento específicas em nome do CONTROLADOR, serão impostas a esse outro operador as mesmas obrigações de proteção de dados estabelecidas neste DPA.

9. TRANSFERÊNCIA INTERNACIONAL DE DADOS

9.1. O OPERADOR não transferirá dados pessoais para fora do Brasil sem a autorização prévia e por escrito do CONTROLADOR.

9.2. Qualquer transferência internacional de dados pessoais deverá cumprir com os requisitos estabelecidos na LGPD.

10. AUDITORIA

10.1. O OPERADOR permitirá e contribuirá para auditorias, incluindo inspeções, realizadas pelo CONTROLADOR ou por outro auditor mandatado pelo CONTROLADOR.

11. PRAZO E RESCISÃO

11.1. Este DPA permanecerá em vigor enquanto o OPERADOR tratar dados pessoais em nome do CONTROLADOR.

11.2. Em caso de rescisão deste DPA, o OPERADOR deverá, a critério do CONTROLADOR, devolver ou eliminar todos os dados pessoais tratados em nome do CONTROLADOR, a menos que a legislação aplicável exija a conservação dos dados pessoais.

12. DISPOSIÇÕES GERAIS

12.1. Este DPA constitui parte integrante do contrato principal celebrado entre as Partes.

12.2. Em caso de conflito entre as disposições deste DPA e as do contrato principal, prevalecerão as disposições deste DPA no que diz respeito ao tratamento de dados pessoais.

12.3. Este DPA será regido e interpretado de acordo com as leis do Brasil.
